# Taksiran Backend API

Node.js Express backend untuk menyimpan dan mengambil domain serta kode toko ke MongoDB.

## Endpoint
- POST /stores — Simpan domain & kode toko
- GET /stores — Ambil semua domain & kode toko

## Konfigurasi
- Edit file `.env` untuk string MongoDB dan port.

## Menjalankan
```
npm install
node index.js
```
